テーマショップ

背景グラデーションテーマの選択・購入（全14テーマ）

v1.1 画面ID: DT-TH-01

目的・役割

無料3種・有料11種の合計14テーマから選択できる画面（v1.0.5 で「ベージュ」を追加し14テーマに）。 有料テーマは 200コイン で購入。動画ボーナスからコインを獲得して購入を促す設計。

UI 構成

要素種類説明
コインバッジHStack現在のコイン残高
動画ボーナスボタンButton+3pt獲得（1日5回まで）
テーマカードLazyVGrid各テーマのプレビュー（グラデーション）
状態バッジOverlay4状態を色別バッジ＋鍵アイコンで表示（下記参照）
購入/適用ボタンButton状態に応じてラベル変化

状態バッジ背景色アイコン

状態

状態表示内容
無料テーマ「適用」ボタン
未購入有料「200コインで購入」ボタン

残高不足購入アラートで警告

使用するデータモデル

AppThemeItem {
    let id: String
    let name: String
    let cost: Int  // 0 or 200
    let colors: [Color]
    let begin: UnitPoint
    let end: UnitPoint
}

ThemeService {
    func getSelectedTheme() -> AppThemeItem
    func setSelectedTheme(id: String)
    func getUnlockedThemeIds() -> [String]
    func purchaseTheme(id: String) -> Bool
}

実装メモ

  • 無料テーマ3種は cost: 0 で常時アンロック
  • 有料11種は購入後に AppGroup に保存（unlocked_themes）
  • v1.0.5 で「ベージュ」テーマを追加（計14テーマ）
  • 適用テーマは ThemeBackgroundView でグラデーション表示
  • 状態バッジは4色（青・緑・グレー・オレンジ）で一目瞭然
  • 動画ボーナスボタンから RewardedAdManager.show() を起動
