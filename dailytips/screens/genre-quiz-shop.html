ジャンルクイズショップ

13ジャンルから興味のあるものをコイン購入で解放（雑学・トリビアは無料）

v1.1 画面ID: DT-GQS-01

目的・役割

13のジャンルから興味のあるものを解放し、ジャンル別クイズで集中学習する画面。 「雑学・トリビア」は完全無料（cost: 0）、その他12ジャンルは最初の5問が無料お試し、全問解放は 100コイン。

使用するデータモデル

GenreDefinition {
    let id: String
    let name: String
    let icon: String  // SF Symbol
    let cost: Int     // 100
}

GenreQuizService {
    func unlockGenre(id: String) -> Bool
    func isUnlocked(id: String) -> Bool
}

実装メモ

  • 13ジャンルの定義は GenreDefinition の static 定数
  • 雑学・トリビアは cost: 0 で常時アンロック扱い
  • お試し5問は解放前にプレビュープレイ（回答記録はしない）。v1.0.7 から1ジャンル1回限り（5問完走時に markFreeTrialUsed(for:) でフラグ記録）
  • v1.0.7: @State playSession: GenrePlaySession? で genre+mode をアトミックに渡し、シート presentation 時の race condition を排除
  • unlockGenre() で CoinService.spendCoins(100) → 成功で AppGroup に保存
  • プレイ開始時に Interstitial広告を挟む（無料ユーザー）
  • 動画ボーナスボタンから RewardedAdManager.show() を起動
  • 実績サマリーの「詳細 ›」は GenreProgressView へ遷移
