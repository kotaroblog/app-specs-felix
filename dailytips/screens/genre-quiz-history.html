ジャンルクイズ履歴

ジャンル別のクイズ回答履歴を確認

v1.0 画面ID: DT-GQS-03

目的・役割

ジャンル別に解いた問題の履歴を時系列で表示。問題と自分の回答・正誤を振り返れる画面。

UI 構成

要素種類説明
ジャンルフィルタPickerすべて / 各ジャンル
履歴リストList日付・問題・正誤マーク

使用するデータモデル

// GenreProgressService が保存する履歴データ
struct GenreQuizHistory {
    let genreId: String
    let questionId: String
    let correct: Bool
    let answeredAt: Date
}

実装メモ

  • AppGroup UserDefaults キー genre_progress_*
  • 履歴件数が多い場合のページネーション（v1.1）
