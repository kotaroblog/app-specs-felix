共有カード（ShareCardView）

豆知識をSNSでシェアするための画像生成View

v1.1 画面ID: DT-HOME-04

目的・役割

豆知識を1080×1080の正方形画像として生成し、X / LINE 等にシェアできるカード。 ImageRenderer でレンダリングし、UIActivityViewController（ShareSheet）に渡す。 ハッシュタグ5種と App Store URL を付与。

UI 構成

非表示Viewとして実装され、レンダリング専用。可視UIは持たない。

要素説明
テーマグラデーション背景選択中テーマ適用
カテゴリバッジtip.category
タイトルtip.title（大きめ）
本文tip.content（要約）
アプリロゴ右下に小さく
ハッシュタグ5種のハッシュタグ（カテゴリ連動）
App StoreリンクテキストURL（ShareService.appStoreURL）

処理フロー

  1. ユーザーがTipCardViewの共有ボタンをタップ
  2. ShareCardView を SwiftUI で構築
  3. ImageRenderer で UIImage に変換
  4. UIActivityViewController で共有シート表示

使用するデータモデル

DailyTip + AppThemeItem
// ShareService.renderCardImage(tip:theme:) -> UIImage

実装メモ

  • ShareService.renderCardImage() で ImageRenderer 使用（1080×1080固定）
  • ShareService.shareText() で X 用テキスト生成（ハッシュタグ5種込み）
  • App Storeリンクは ShareService.appStoreURL 定数
  • Instagram Stories 対応は削除（仕様変更により動作不可のため廃止）
