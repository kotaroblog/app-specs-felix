ジャンルクイズプレイ画面

ジャンル別の問題を解くプレイ画面（3モード対応）

v1.1 画面ID: DT-GQS-02

目的・役割

解放済みジャンルの問題を4つのプレイモード（ランダム / 順番通り / 続きから / お試し）で解いていく画面。

プレイモード

モード動作進捗記録
ランダム未回答問題からランダムに出題あり
順番通りposition をリセットして1問目からあり
続きから保存された position から再開あり
お試し（v1.1）未解放ジャンルで最初の5問のみ出題なし（解放後に改めて記録）

お試しモード終了画面（v1.1）

お試し5問終了時、「お試し終了！」ヘッダーと「残り95問を解放しよう！」誘導カード（オレンジ→ピンクのグラデーション）を表示する。

  • 「\(genre.cost)ptで全問解放、または毎日のptを貯めて挑戦できます」の文言
  • アイコンは sparkles（trophy ではない）

要素種類説明

状態

遷移

操作遷移先
選択肢タップ同画面（結果表示）
「次へ」同画面（次問題）
終了GenreQuizShopView へ戻る

使用するデータモデル

GenreQuizQuestion {
    let id: String
    let question: String
    let choices: [String]
    let correct: Int
    let explanation: String?
}

GenreProgressService {
    func recordAnswer(genreId: String, questionId: String, correct: Bool)
    func getPosition(genreId: String) -> Int
}

実装メモ

  • プレイ開始時にInterstitial広告（無料ユーザー）
  • 回答ごとに GenreProgressService.recordAnswer() で進捗保存
  • 「続きから」モードは position を維持
