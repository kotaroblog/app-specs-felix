通知機能

毎日の豆知識リマインドをローカル通知で配信

v1.1 機能ID: DT-FEAT-NOTIF

概要

UserNotifications フレームワーク（ローカル通知）を使い、ユーザーが設定した時刻に「今日の豆知識」リマインダーを毎日配信。 プッシュ通知（APNs）は使用せず、サーバー不要のローカル完結型。

関連画面

  • 設定画面（通知ON/OFF・時刻設定・復習通知ON/OFF）

処理フロー

  1. 初回 ON 時、通知許可をリクエスト
  2. 許可されたら指定時刻で UNCalendarNotificationTrigger をスケジュール
  3. 毎日その時刻にリマインダー表示
  4. OFF 時、すべての通知を removeAllPendingNotificationRequests でキャンセル
  5. 時刻変更時は再スケジュール

API

@MainActor
final class NotificationService {
    func setEnabled(_ enabled: Bool, at time: Date)
    func setTime(_ time: Date)
    func getEnabled() -> Bool
    func scheduleDailyNotification(at time: Date)
}

通知内容

項目内容
title「今日の豆知識」
body「新しい知識をチェックしよう」 or その日の tip タイトル
sounddefault
repeatstrue（毎日）

ビジネスルール

  • 初回ON時のみ通知許可ダイアログを表示
  • 許可拒否されたら設定アプリへの誘導アラート
  • 時刻はユーザー設定可（デフォルトは8時）
  • 通知タップでアプリ起動 → HomeView
  • 権限拒否時は設定アプリへの誘導アラートを表示

外部連携

連携先用途
UserNotificationsローカル通知のスケジューリング
Info.plist通知許可ダイアログの表示文言

エラー処理

発生条件対応
通知許可拒否「設定アプリで通知を有効に」アラート + 設定アプリ起動誘導
scheduleDailyNotification 失敗Logger.error、UI 上は無視

復習通知（忘却曲線・v1.1）

毎日の通知に加え、「復習」用の通知を別途スケジュールする。エビングハウスの忘却曲線に基づき、 1日後・3日後・7日後の夜21時に「復習タイム」リマインダーを送信。

項目内容
識別子review_reminder_1 / review_reminder_3 / review_reminder_7
送信時刻各オフセット日数の 21:00 固定
title「復習タイム」
body「N日前の豆知識を覚えていますか？タップして思い出しましょう」
repeatsfalse（一度きり・有効化のたびに再スケジュール）
ON/OFF設定画面で個別トグル（毎日通知が有効な場合のみ操作可能）
// NotificationService に追加
func getReviewEnabled() -> Bool
func setReviewEnabled(_ value: Bool)
func scheduleReviewNotifications() async  // 1/3/7日後 21:00
func cancelReviewNotifications()

UserDefaults キー: notification_review_enabled

実装メモ

  • NotificationService は @MainActor
  • UserDefaults キー: notification_enabled, notification_hour, notification_minute, notification_review_enabled
  • プッシュ通知（APNs）は用途なし → ローカル通知のみ
  • 復習通知識別子は review_reminder_ プレフィックスで管理
